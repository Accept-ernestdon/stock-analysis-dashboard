# 📊 Financial Market Analysis Dashboard

## 📌 Overview
This project is an interactive financial data dashboard built using Python and Streamlit. It allows users to analyze real-world stock data, visualize trends, and assess investment risk using key financial metrics.

The application is deployed online and accessible from anywhere.

---

## 🌐 Live Demo
👉 https://stock-analysis-dashboard-pte4wrsemwxduh6hgidtg7.streamlit.app/

---

## 🚀 Features
- 📈 Multi-stock comparison (AAPL, MSFT, TSLA)
- 📊 Interactive price trend visualization
- 📉 Moving averages (20-day & 50-day)
- 📉 Daily returns analysis
- ⚠️ Volatility (risk) measurement
- 🔥 Correlation heatmap between stocks
- 📁 Download stock data as CSV

---

## 🛠️ Tools & Technologies
- Python
- Streamlit
- Pandas
- Matplotlib
- Yahoo Finance API (yFinance)
